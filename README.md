# trader-sentiment-analysis

---

## Overview

Analyzes how Bitcoin market sentiment (Fear/Greed Index) relates to trader behavior and performance on Hyperliquid. Uncovers patterns across 211,000+ trades to propose deployable trading strategies.

---

## Setup

### Requirements
- Python 3.9+
- Jupyter Notebook or JupyterLab

### Install dependencies

```bash
pip install pandas numpy matplotlib seaborn scipy scikit-learn jupyter
```

---

## How to Run

```bash
jupyter notebook trader_sentiment_analysis_1.ipynb
```

Then **Run All Cells** (`Kernel → Restart & Run All`).

Output charts are saved automatically to a `charts/` folder in the same directory.
